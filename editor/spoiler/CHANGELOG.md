# Changelog

## [1.0.0] - 2026-05-16

### Added
- Initial release of Spoiler Block extension
- Collapsible spoiler blocks with smooth animations
- Click to reveal/hide spoiler content
- Dark mode support with proper styling
- Responsive design for mobile and desktop
- SVG icon that rotates on expand/collapse

### Features
- Simple `:::spoiler` syntax for easy use
- Hover effects on summary for better UX
- Prose styling for content inside spoilers
- Border and background styling for visual appeal
